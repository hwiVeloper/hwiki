# hwiki Query

위키에 대해 질문하고 답변을 생성한다.

## 사용법

```
/hwiki-query <질문>
```

- 인자 없이 실행하면 사용자에게 질문을 물어본다.

## 워크플로우

1. `$ARGUMENTS`에서 질문을 파싱한다.
2. `wiki/index.md`를 읽어 관련 페이지를 파악한다.
3. 관련 페이지들을 읽고 답변을 생성한다.
4. 답변이 위키 페이지로 저장할 가치가 있으면 사용자에게 제안한다:
   - "이 답변을 위키 페이지로 저장할까요?"
5. 사용자가 동의하면:
   - synthesis 페이지를 생성한다.
   - `wiki/index.md`를 업데이트한다.
   - `wiki/log.md`에 `## [YYYY-MM-DD] query | 질문 주제` 형식으로 기록을 추가한다.
   - git commit한다. 메시지: `query: [질문 주제]`

## 위키 규칙 (CLAUDE.md 참조)

- 모든 위키 페이지는 한국어로 작성한다.
- 파일명은 영문 kebab-case
- 교차참조는 `[[wikilink]]` 스타일을 사용한다.
- 모든 위키 페이지는 `wiki/` 아래 flat 구조로 위치한다.
